# Sitio Web Ana Albiol

Este es el sitio web personal de Ana Albiol, desarrollado con HTML, CSS y JavaScript vanilla.

## Estructura

- `index.html`: Página principal.
- `css/style.css`: Estilos del sitio.
- `js/main.js`: Lógica y animaciones.
- `assets/`: Imágenes y recursos.

## Cómo ver la web

Simplemente abre el archivo `index.html` en tu navegador web de preferencia.
